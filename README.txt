Long Hoang Logistics - Static website (Melody-like design)
Files included:
- index.html, about.html, sea.html, air.html, road.html, customs.html, contact.html, news.html
- extra service pages: overseas.html, duan.html, doortodoor.html
- css/style.css
- js/main.js
- images/logo.png (placeholder SVG) - replace with your PNG
How to use:
1. Unzip and upload all files to your hosting (or GitHub Pages).
2. Replace images in images/ with your actual images, and logo.png with your logo.
3. Edit HTML files to update text and contact details.
4. Update footer social links for Facebook/Zalo.
